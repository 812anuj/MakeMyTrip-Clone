# ✈️ MakeMyTrip Clone – Full-Stack Travel Booking App

A full-featured **MakeMyTrip-inspired clone** developed with **Next.js (React)** on the frontend and **Spring Boot (Java)** with **MongoDB** on the backend.

It supports **flight and hotel bookings**, email confirmations with **PDF e-tickets**, an **admin dashboard**, and a smart **refund system**.

---

## 🔧 Tech Stack

| Frontend   | Backend      | Database | Tools       |
|------------|--------------|----------|-------------|
| Next.js 14 | Spring Boot  | MongoDB  | iText PDF   |
| Tailwind   | Java 17      | Mongoose | JavaMail API|
| Axios      | REST APIs    |          |             |

---

## ✨ Features

- 🔍 **Search Flights & Hotels** by city
- 🛫 **Book Flights** with seat availability
- 🏨 **Book Hotels** with room selection
- 📧 **Email Confirmation** with attached PDF ticket
- 💸 **Smart Refund Logic:**
  - ✅ 100% refund if canceled ≥48h before departure
  - ✅ 50% refund if canceled ≥24h
  - ❌ No refund if canceled <24h before
- 🔐 **JWT-based Auth**
- 🧾 **Admin Panel** to manage flights and hotels
- 🧑‍💼 User dashboard with upcoming/cancelled bookings
- 📱 **Responsive UI** (mobile-friendly)
